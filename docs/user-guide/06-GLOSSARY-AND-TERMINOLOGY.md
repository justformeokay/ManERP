| Shortcut / Tip | EN | ID | KO | ZH |
|---|---|---|---|---|

| Dashboard Refresh | Dashboard widgets refresh automatically; use browser refresh for immediate update | Widget dasbor diperbarui secara otomatis; gunakan refresh browser untuk pembaruan segera | 대시보드 위젯은 자동 새로고침됩니다; 즉시 업데이트하려면 브라우저를 새로고침하세요 | 仪表板组件自动刷新；使用浏览器刷新立即更新 |
